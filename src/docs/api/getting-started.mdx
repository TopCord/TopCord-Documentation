# Getting Started
The basics of the rest API.

### Reference Guide

- **Base URL:** https://api.topcord.xyz/
- **API Token:** This can be obtained at your bots edit page.

### Authentication 

Some API endpoints will require you to authenticate using the `Authorization` header, an example can be seen below.

```json title=Headers lineNumbers=true
{
  "Authorization": "token"
}
```
